# SIGEL - Sistema de Gestión Electoral

Este proyecto es una plataforma integral de inteligencia y gestión electoral (SIGEL) que incluye un backend robusto en Python y un frontend moderno en React.

## Estructura del Proyecto

- `backend/`: API, lógica de procesamiento de datos e inteligencia electoral (Python/FastAPI).
- `frontend/`: Interfaz de usuario interactiva y mapas (React + Leaflet).
- `docs/`: Documentación técnica y guías detalladas.

## Requisitos

### Backend
- Python 3.11+
- Ver `backend/requirements.txt` para las dependencias necesarias.

### Frontend
- Node.js (v18+)
- NPM / Yarn
