# Sistema de Catálogos

Este proyecto es un sistema de gestión de catálogos que incluye un backend en Python y un frontend en React.

## Estructura del Proyecto

- `backend/`: API y lógica de procesamiento de datos (Python).
- `frontend/`: Interfaz de usuario (React + Leaflet).
- `docs/`: Documentación técnica y guía de iniciación.

## Requisitos

### Backend
- Python 3.x
- Ver `backend/requirements.txt` (si existe) o las dependencias instaladas en `.venv`.

### Frontend
- Node.js
- NPM / Yarn
