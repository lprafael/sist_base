Sistema Base - Poliverso

Resumen Ejecutivo

El Sistema Base - Poliverso es una solución estructural integral para el inicio de proyectos web, proporcionando gestión de usuarios avanzada, roles, seguridad reactiva, auditoría completa y herramientas de administración del sistema.

Arquitectura y Tecnologías

ComponenteTecnología
FrontendReact 18, Vite, CSS Custom, JWT
BackendFastAPI (Python 3.11+), SQLAlchemy Async, PostgreSQL
Base de DatosPostgreSQL 14+, SQLAlchemy ORM
EmailSMTP seguro (notificaciones y contraseñas)
Frontend → Backend → DB

Funcionalidades Core

  • Gestión Completa de Usuarios: Alta, Edición, Activación, Desactivación, Reactivación y Borrado Físico/Lógico.
  • Sistema de Roles y Permisos (RBAC): Control granular de acceso a nivel de API e interfaz.
  • Seguridad de Acceso: Autenticación via JWT con flujos de recuperación y generación de contraseñas temporales por el administrador.
  • Auditoría Integrada: Registro detallado de cada acción del usuario, IP, dispositivo y cambios realizados.
  • Administración del Sistema: Configuración de parámetros dinámicos y sistema de backup automático de tablas core.
  • Notificaciones: Sistema de notificaciones por email para eventos críticos de seguridad.

Gestión de Contraseñas

  1. Recuperación: El usuario puede solicitar la recuperación de su cuenta ante un olvido.
  2. Generación por Admin: El administrador tiene la potestad de generar una nueva contraseña temporal segura y enviarla directamente al usuario desde la interfaz.
  3. Cambio de Password: El usuario puede actualizar su contraseña en cualquier momento desde su perfil una vez autenticado.

Seguridad y Buenas Prácticas

  • Hash de contraseñas mediante algoritmo Bcrypt.
  • Implementación de CORS y Token Bearer para comunicación segura entre capas.
  • Auditoría inmutable de transacciones.
  • Despliegue modular compatible con Docker para alta escalabilidad.
